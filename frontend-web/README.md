# Frontend Web - Aplikasi Servis Alat Elektronik

Frontend web untuk aplikasi servis alat elektronik on-demand menggunakan React + Vite + Tailwind CSS.

## Fitur

- ✅ Authentication (Login, Register Customer, Register Technician)
- ✅ Pencarian Teknisi dengan Filter (Kategori, Lokasi, Rating)
- ✅ Manajemen Order (Create, Accept, Reject, Start, Complete)
- ✅ Chat antara Customer dan Technician
- ✅ Negosiasi Harga (Tawar-menawar)
- ✅ Pembayaran
- ✅ Review & Rating
- ✅ Riwayat Order & Invoice
- ✅ Notifikasi
- ✅ Admin Panel

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Setup

1. Pastikan backend API sudah running di `http://localhost:8000`
2. Frontend akan berjalan di `http://localhost:3000`
3. API calls akan di-proxy ke backend melalui Vite proxy

## Tech Stack

- **React 18** - UI Library
- **Vite** - Build Tool
- **React Router** - Routing
- **Axios** - HTTP Client
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Structure

```
src/
├── components/     # Reusable components
├── pages/         # Page components
├── contexts/      # React Context (Auth, etc)
├── services/      # API services
├── utils/         # Utility functions
├── App.jsx        # Main App component
└── main.jsx       # Entry point
```
